# Animation Skipper
Fades screen to black during skilling animations to improve awareness.
This is helpful for AFK activities so it is more obvious when your character stops performing actions.

Supports:
- Cooking
- Firemaking
- Fishing
- Mining
- Woodcutting
- Fletching
- Herblore
- Crafting
- Smithing

Based on the works of [Skilling Notifications](https://github.com/erwinvc/runelite-plugins/tree/skilling-notifications)
