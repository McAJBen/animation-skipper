# Animation Skipper
Skips player animations while skilling to improve training efficiency

Supports:
- Cooking
- Firemaking
- Fishing
- Mining
- Woodcutting
- Fletching
- Herblore
- Crafting
- Smithing

Based on the works of [Skilling Notifications](https://github.com/erwinvc/runelite-plugins/tree/skilling-notifications)
